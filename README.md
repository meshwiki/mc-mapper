# MC-Mapper

MC-Mapper is part of the [meshcore digital twin project](https://github.com/meshwiki/digitwin)

This project was heavily inspired by https://mapme.sh and collects raw statistical data from a meshcore companion using [meshcore.js](https://github.com/meshcore-dev/meshcore.js)

## Goals

The goal of this project is to collect realworld reception data combined with location data. We aim to map repeater signal profiles and overlap and interference between them. We will then try to make predictions on how changing repeater settings or characteristics (like tx, antenna gain and interference threshold) benefit or degrade mesh behaviour.
